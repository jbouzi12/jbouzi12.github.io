Although I've had a unique undergraduate experience, seemingly unrelated to computer programming, I've had a passion for technology ever since joining an after school engineering program in the 7th grade.

Recent Projects

PinPoint

PinPoint is a Chrome Extension that allows you to take notes on Youtube videos in real time and keep track of your annotated videos. Each note automatically has a time associated with it, allowing the user to easily jump to key moments in a video, and is saved and readily accessible for future reference.

Built with: JavaScript, Google-Chrome API

GameOverflow

GameOverflow is a web application that revolves around gaming. Users can create a profile, post articles on any game, genre, or console, and comment as well as vote on other user's content. Along with the Ruby on Rails backend, GameOverflow was built using Test Driven Development via an RSpec test suite, an ActiveRecord database, and HTML and CSS for our web design.

Built with: Ruby, Rails, TDD, RSpec, PostgreSQL, CSS

Pair Me Up

Pair Me Up is a Chrome Extension built to optimize the pair programming experience for students. Users can use the extension to coordinate roles during a pairing session, give eachother feedback, time their session and log their times in both navigator and driver roles, and keep track of their skills in both of these roles.

Built with: Sinatra, Ruby, CSS, JavaScript, AJAX, jQuery, ActiveRecord, Google Chrome API

MiddleGround

MiddleGround is an app that helps two friends find interesting places to meet at a place between them so neither has to travel longer than the other. This app is unique because we find the midpoint based on travel time, not distance.

Built with: Ruby, Google Maps API, Yelp API, Sinatra, jQuery, AJAX
